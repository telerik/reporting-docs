# The `scale` option of the Angular Report Viewer

Sets the scale factor for the report pages. Takes effect when scaleMode is set to "SPECIFIC".

## Type

`number`

## Examples

````html
<tr-viewer
  [scale]="scale">
</tr-viewer>
````

````typescript
scale: number = 1.5;
````

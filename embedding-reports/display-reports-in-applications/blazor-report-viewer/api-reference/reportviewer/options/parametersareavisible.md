# The `ParametersAreaVisible` option of the Blazor Report Viewer

Gets or sets a value indicating whether the viewer's parameters area is initially visible.

## Type

`System.Boolean`

## Examples

````C#
<ReportViewer ParametersAreaVisible="true" />
````

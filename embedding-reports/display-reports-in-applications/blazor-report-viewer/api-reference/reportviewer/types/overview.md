# Types of the Blazor Report Viewer

| Type | Description |
| ------ | ------ |
| [AccessibilityKeyMap]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/accessibilitykeymap%})|Provides a class for the key mapping used when report viewer accessibility is enabled.|
| [ClientEventsOptions]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/clienteventsoptions%})|Specifies the viewer's event options. Each option can be assigned an event handler name.<br>The JavaScript API of the event handlers is available in <br>https://docs.telerik.com/reporting/html5-report-viewer-reportviewer-events-ready.|
| [Command]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/command%})|Specifies a report viewer command.|
| [CommandBase]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/commandbase%})|Specifies a report viewer command.|
| [CommandOptions]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/commandoptions%})|Exposes the available report viewer commands.|
| [EditorsOptions]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/editorsoptions%})|Specifies the settings for the different types of configurable report parameter editors.|
| [ExportCommand]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/exportcommand%})|Specifies a report viewer command.|
| [GoToPageCommand]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/gotopagecommand%})|Specifies a report viewer command.|
| [ParameterEditor]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/parametereditor%})|Specifies the parameter editor functions.|
| [ParametersOptions]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/parametersoptions%})|Specifies the options for the two parameter editors types - `Telerik.ReportViewer.Blazor.EditorsOptions.SingleSelect` and `Telerik.ReportViewer.Blazor.EditorsOptions.MultiSelect`.|
| [ReportServerOptions]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/reportserveroptions%})|Specifies the configuration details for connecting to a Telerik Report Server instance.<br>Report Server for .NET supports Token (GetPersonalAccessToken) or username/password authentication.<br>Report Server for .NET Framework 4.6.2 supports username/password or Guest account (Url only, if enabled).|
| [ReportSourceOptions]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/reportsourceoptions%})|Specifies the report and initial report parameter values to be displayed in the report viewer.|
| [SendEmailOptions]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/sendemailoptions%})|Specifies the send email option settings.|
| [ZoomCommand]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/blazor-report-viewer/api-reference/reportviewer/types/zoomcommand%})|Specifies a report viewer command.|

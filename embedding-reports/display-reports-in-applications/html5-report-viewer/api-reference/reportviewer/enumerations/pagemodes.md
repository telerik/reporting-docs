# The `PageModes` enumeration of the HTML5 Report Viewer

Enum for the available page modes in the report viewer.

## Values

| Value | Description |
| ------ | ------ |
| SINGLE_PAGE | Display one page at a time. |
| CONTINUOUS_SCROLL | Display pages in a continuous scroll. |

# Enumerations of the HTML5 ASP.NET MVC Report Viewer

| Enumeration | Description |
| ------ | ------ |
| [DocumentMapAreaPosition]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/enumerations/documentmapareaposition%})|Specifies the positioning options for the Document Map Area within the MVC Report Viewer interface.<br>Used to configure the placement of the document map navigation panel relative to the report content area.<br>The Document Map Area provides hierarchical navigation through report bookmarks and sections.|
| [EditorTypes]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/enumerations/editortypes%})|Specifies the available parameter editor types for rendering parameter input controls in the MVC Report Viewer.<br>Used to define the visual style and interaction behavior of parameter input controls.<br>Each editor type provides different user experience and visual presentation for parameter value selection.|
| [PageMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/enumerations/pagemode%})|Specifies the page display mode options for the MVC Report Viewer control.<br>Used to configure how report pages are presented and navigated within the viewer interface.<br>Controls the page rendering behavior and user navigation experience within the report viewer.|
| [ParametersAreaPosition]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/enumerations/parametersareaposition%})|Specifies the positioning options for the Parameters Area within the MVC Report Viewer interface.<br>Used to configure the placement of the parameters panel relative to the report content area for optimal user experience and interface layout.<br>Controls the visual placement of the parameters input panel to accommodate different screen sizes and layout preferences.|
| [PrintMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/enumerations/printmode%})|Specifies the printing method options for the MVC Report Viewer control.<br>Used to configure how the report viewer handles print operations and determines the optimal printing mechanism based on browser capabilities and requirements.<br>Controls the printing behavior to ensure optimal print experience across different browsers and environments.|
| [ScaleMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/enumerations/scalemode%})|Specifies the scale mode options for the MVC Report Viewer control to determine how reports are sized within the viewer area.<br>Used to configure the automatic sizing behavior and zoom level management for optimal report display and readability.<br>Controls the report sizing strategy to optimize readability and user experience based on viewer dimensions and content requirements.|
| [ViewMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/enumerations/viewmode%})|Specifies the display mode options for the MVC Report Viewer control to determine how reports are presented and how users interact with them.<br>Used to configure whether the viewer displays reports in interactive mode with full functionality or print preview mode optimized for printing.<br>Controls the report presentation style and interaction capabilities to optimize user experience for different viewing scenarios.|

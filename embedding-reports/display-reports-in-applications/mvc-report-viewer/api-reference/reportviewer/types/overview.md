# Types of the HTML5 ASP.NET MVC Report Viewer

| Type | Description |
| ------ | ------ |
| [AccessibilityKeyMap]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/accessibilitykeymap%})|Provides a class for the key mapping used when report viewer accessibility is enabled.|
| [CustomParameterEditor]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/customparametereditor%})|Represents a custom parameter editor configuration for the MVC Report Viewer, enabling custom UI components for report parameter input.<br>Both functions are executed in the client-side JavaScript context of the MVC Report Viewer.|
| [DeferredScripts]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/deferredscripts%})|Represents a container for deferred JavaScript initialization statements for MVC Report Viewer components.<br>Used to collect and render JavaScript initialization code at a specific location in the HTML document for optimal performance and loading behavior.<br>Implements IHtmlString to enable direct rendering in MVC views and provides control over script tag generation for flexible script placement.|
| [Editors]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/editors%})|Provides configuration options for parameter editor types used in the MVC Report Viewer.<br>Used to specify which input control types should be used for single-select and multi-select parameter scenarios.<br>Controls the visual appearance and interaction behavior of parameter input controls based on parameter configuration.|
| [HtmlHelpers]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/htmlhelpers%})|A class that contains a collection of all Telerik Reporting widgets|
| [InstanceReportSource]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/instancereportsource%})|This report source is used when you have a reference to an already instantiated report <br>(`Telerik.Reporting.Report` instance). For better performance and compatibility<br>use [`Telerik.Reporting.TypeReportSource`]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/typereportsource%}). <br>Represents a report source that contains a reference to an instance of <br>`Telerik.Reporting.IReportDocument`.|
| [Parameters]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/parameters%})|Provides configuration options for report parameters display and behavior within the MVC Report Viewer.<br>Used to customize how parameter input controls are rendered and how users interact with report parameters.<br>Contains editor configuration settings that control the appearance and functionality of parameter input controls.|
| [ReportServer]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/reportserver%})|Provides configuration data for connecting to a Telerik Report Server instance from the MVC Report Viewer.<br>Used to establish authenticated connections to Report Server for centralized report management and deployment scenarios.<br>Connecting to Telerik Report Server for .NET is not supported.<br>Contains connection details including server URL and authentication credentials for Report Server integration.|
| [ReportSource]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/reportsource%})|Provides the base class for a report source. This is an abstract class<br>that contains only a collection of parameter values that are passed to the<br>`Telerik.Reporting.IReportDocument` object when its instance is initialized.|
| [ReportSourceCollection]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/reportsourcecollection%})|Represents a collection of <br>[`ReportSource`]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/methods/reportsource%}) objects.|
| [SendEmail]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/sendemail%})|Provides a class for the send e-mail message options.<br>Contains the enabled property, the preselected rendering Format, from and to e-mail addresses.|
| [TypeReportSource]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/typereportsource%})|Represents a report source that allows a report document to be instantiated from an `System.Type.AssemblyQualifiedName`.|
| [UriReportSource]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/urireportsource%})|Represents a report source that loads a report document from a URI.<br>The URI must point to a physical TRDX, TRDP or TRBP file.|
| [XmlReportSource]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/mvc-report-viewer/api-reference/reportviewer/types/xmlreportsource%})|Represents a report source that contains the XML markup of a report document.|

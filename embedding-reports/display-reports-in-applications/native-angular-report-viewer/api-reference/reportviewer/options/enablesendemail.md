# The `enableSendEmail` option of the Native Angular Report Viewer

Enables or disables the send email functionality.

## Type

`boolean`

## Examples

````html
<reporting-angular-viewer
 [enableSendEmail]="enableSendEmail">
</reporting-angular-viewer>
````

````typescript
enableSendEmail: boolean = true;
````

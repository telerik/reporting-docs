# The `height` option of the Native Angular Report Viewer

Sets the height of the report viewer container.

## Type

`string`

## Examples

````html
<reporting-angular-viewer #viewerId
 [height]="height">
</reporting-angular-viewer>
````

````typescript
height: string = '600px';
````

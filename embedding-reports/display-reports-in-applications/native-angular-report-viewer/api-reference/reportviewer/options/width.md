# The `width` option of the Native Angular Report Viewer

Sets the width of the report viewer container.

## Type

`string`

## Examples

````html
<reporting-angular-viewer
 [width]="width">
</reporting-angular-viewer>
````

````typescript
width: string = '800px';
````

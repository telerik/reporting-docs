# The `ExportFormat` type of the Native Angular Report Viewer

Export format options for the Native Angular Report Viewer.
The available export options are:

- 'pdf'
- 'csv'
- 'xlsx'
- 'pptx'
- 'rtf'
- 'image'
- 'docx'

## Type Definition

````typescript
type ExportFormat = "pdf" | "csv" | "xlsx" | "pptx" | "rtf" | "image" | "docx";
````

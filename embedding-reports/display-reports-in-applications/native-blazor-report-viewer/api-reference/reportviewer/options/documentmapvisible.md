# The `DocumentMapVisible` option of the Native Blazor Report Viewer

Gets or sets  a value indicating whether the viewer's document map is initially visible.

## Type

`System.Boolean`

## Examples

````C#
<ReportViewer 
    DocumentMapVisible="true">
</ReportViewer>
````

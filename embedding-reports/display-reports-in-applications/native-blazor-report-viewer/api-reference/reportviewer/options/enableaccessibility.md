# The `EnableAccessibility` option of the Native Blazor Report Viewer

Enables or disables the accessibility features of the report viewer and its contents.

## Type

`System.Boolean`

## Examples

````C#
<ReportViewer 
    EnableAccessibility="true">
</ReportViewer>
````

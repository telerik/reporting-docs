# Options of the Native Blazor Report Viewer

| Property | Description |
| ------ | ------ |
| [AuthenticationToken]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/authenticationtoken%})|Gets or sets the authentication (bearer) token that is passed in the Authorization header of the viewer's requests.|
| [DocumentMapVisible]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/documentmapvisible%})|Gets or sets  a value indicating whether the viewer's document map is initially visible.|
| [EnableAccessibility]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/enableaccessibility%})|Enables or disables the accessibility features of the report viewer and its contents.|
| [EnableSendEmail]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/enablesendemail%})|Determines whether the send Email functionality is enabled. If set to false, the send Email button will not be rendered in the toolbar.|
| [Height]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/height%})|The width of the component.|
| [KeepClientAlive]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/keepclientalive%})|Gets or sets a value indicating whether the client will be kept alive.<br><br> When set to true a request will be sent to the server to stop the client from expiring, determined by the ClientSessionTimeout server configuration<br><br> When set to false, the client will be left to be expired|
| [PageMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/pagemode%})|Sets if the report is displayed in Single page or Continuous scroll mode.|
| [ParameterBooleanTemplate]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/parameterbooleantemplate%})|Template used to render boolean report viewer parameter editors.<br>Allows for declaring custom boolean report viewer parameter editors.|
| [ParameterDateTimeTemplate]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/parameterdatetimetemplate%})|Template used to render DateTime report viewer parameter editors.<br>Allows for declaring custom DateTime report viewer parameter editors.|
| [ParameterDoubleTemplate]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/parameterdoubletemplate%})|Template used to render double report viewer parameter editors.<br>Allows for declaring custom double report viewer parameter editors.|
| [ParameterIntegerTemplate]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/parameterintegertemplate%})|Template used to render integer report viewer parameter editors.<br>Allows for declaring custom integer report viewer parameter editors.|
| [ParameterMultiSelectTemplate]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/parametermultiselecttemplate%})|Template used to render multi-select report viewer parameter editors.<br>Allows for declaring custom multi-select report viewer parameter editors.|
| [ParametersAreaVisible]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/parametersareavisible%})|Gets or sets a value indicating whether the viewer's parameters area is initially visible.|
| [ParameterSingleSelectTemplate]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/parametersingleselecttemplate%})|Template used to render single-select report viewer parameter editors.<br>Allows for declaring custom single-select report viewer parameter editors.|
| [ParameterStringTemplate]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/parameterstringtemplate%})|Template used to render string report viewer parameter editors.<br>Allows for declaring custom string report viewer parameter editors.|
| [ParameterTemplate]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/parametertemplate%})|Generic template used to render report viewer parameter editors.<br>Allows for declaring custom report viewer parameter editors.|
| [PrintMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/printmode%})|Gets or sets the print mode of the viewer.<br>Three modes exist currently:<br>AutoSelect - The viewer automatically determinse how to print: via the browser's PDF plug-in or exporting to a PDF file.<br>ForcePDFPlugin - The viewer always uses the PDF plugin regardless of the browser's version and settings.<br>ForcePDFFile - The viewer always exports the report document to PDF format with 'automatic print on open' script enabled.|
| [ReportSource]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/reportsource%})|Gets or sets the [`Telerik.ReportViewer.BlazorNative.ReportSourceOptions`]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/types/reportsourceoptions%})<br>used to determine which report will be displayed by the viewer.|
| [ReportViewerSettings]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/reportviewersettings%})|Defines a renderfragment that contains all settings for the Report Viewer.|
| [Scale]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/scale%})|Gets or sets zoom ratio of the report.<br>1.0 is equal to 100%, i.e. the original size of the report.|
| [ScaleMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/scalemode%})|Gets or sets the scale mode of the viewer.<br>Three modes exist currently:<br>FitPage - The whole report will fit on the page (will zoom in or out), regardless of its width and height.<br>FitPageWidth - The report will be zoomed in or out so that the width of the screen and the width of the report match.<br>Specific - Uses the scale to zoom the report in and out.|
| [ServiceType]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/servicetype%})|Specifies the type of service the ReportViewer is using.|
| [ServiceUrl]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/serviceurl%})|Gets or sets the URL of the service which will provide the report viewer with reports<br>(relative or absolute).<br>The service must be properly configured so that the report viewer can<br>successfully communicate with the server.<br>For more information on how to configure the service please check<br>http://www.telerik.com/help/reporting/telerik-reporting-rest-conception.html.|
| [Tools]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/tools%})|Represents the collection of tools available in the report viewer's toolbar.<br>Allows for customizing which tools from the viewer's toolbar are visible and which ones are hidden.|
| [TooltipTemplate]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/tooltiptemplate%})|Gets or sets the template for rendering tooltips in the ReportViewer.|
| [ViewMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/viewmode%})|Gets or sets the view mode.<br>PrintPreview - Displays the paginated report as if it is printed on paper. Interactivity is disabled.<br>Interactive - Displays the report in its original width and height with no paging. Interactivity is enabled.|
| [Width]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-blazor-report-viewer/api-reference/reportviewer/options/width%})|The width of the component.|

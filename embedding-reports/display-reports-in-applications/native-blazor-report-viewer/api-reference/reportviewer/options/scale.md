# The `Scale` option of the Native Blazor Report Viewer

Gets or sets zoom ratio of the report. 1.0 is equal to 100%, i.e. the original size of the report.

## Type

`System.Double`

## Examples

````C#
<ReportViewer 
    Scale="1.5">
</ReportViewer>
````

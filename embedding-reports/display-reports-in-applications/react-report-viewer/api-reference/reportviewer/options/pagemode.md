# The `pageMode` option of the React Report Viewer

Sets if the report is displayed in Single page or Continuous scroll mode. Available values: "SINGLE_PAGE" (only one page loaded), "CONTINUOUS_SCROLL" (multiple pages loaded). Default value is "CONTINUOUS_SCROLL".

## Type

`string`

## Examples

````JavaScript
// Set page mode to single page<TelerikReportViewer  serviceUrl="api/reports/"  reportSource={{ report: "Dashboard.trdp" }}  pageMode="SINGLE_PAGE"/>
````

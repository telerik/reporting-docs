# The `searchMetadataOnDemand` option of the React Report Viewer

Determines whether the search metadata will be delivered on demand (true) or by default (false). Default value is false.

## Type

`boolean`

## Examples

````JavaScript
// Enable search metadata on demand<TelerikReportViewer  serviceUrl="api/reports/"  reportSource={{ report: "Dashboard.trdp" }}  searchMetadataOnDemand={true}/>
````

# parametersAreaPosition

Specifies where the Parameters Area should be displayed. Available values: "RIGHT", "TOP", "LEFT", "BOTTOM".

## Type

`string`

## Examples

````html
<tr-viewer
  [parametersAreaPosition]="parametersAreaPosition">
</tr-viewer>
````

````typescript
parametersAreaPosition: string = "LEFT";
````

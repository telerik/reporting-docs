# parametersAreaVisible

Determines whether the viewer's parameters area is displayed by default when parameter editors exist.

## Type

`boolean`

## Examples

````html
<tr-viewer
  [parametersAreaVisible]="parametersAreaVisible">
</tr-viewer>
````

````typescript
parametersAreaVisible: boolean = false;
````

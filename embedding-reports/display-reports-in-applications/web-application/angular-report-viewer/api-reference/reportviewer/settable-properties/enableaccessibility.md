# enableAccessibility

Determines whether the viewer should provide accessibility features support.

## Type

`boolean`

## Examples

````html
<tr-viewer
  [enableAccessibility]="enableAccessibility">
</tr-viewer>
````

````typescript
enableAccessibility: boolean = true;
````

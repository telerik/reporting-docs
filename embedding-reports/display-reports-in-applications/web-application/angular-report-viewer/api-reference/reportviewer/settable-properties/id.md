# id

Sets the unique identifier of the ReportViewer instance. If not specified, the id of the target HTML element will be used. The id is used to identify the client session when persistSession is enabled.

## Type

`string`

## Examples

````html
<tr-viewer #viewerId>
</tr-viewer>
````

# searchMetadataOnDemand

Determines whether search metadata will be delivered on demand (true) or by default (false).

## Type

`boolean`

## Examples

````html
<tr-viewer
  [searchMetadataOnDemand]="searchMetadataOnDemand">
</tr-viewer>
````

````typescript
searchMetadataOnDemand: boolean = true;
````

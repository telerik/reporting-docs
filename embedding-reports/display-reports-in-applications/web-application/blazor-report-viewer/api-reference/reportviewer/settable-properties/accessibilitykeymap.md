# AccessibilityKeyMap

Gets or sets the key shortcuts map, used when the report viewer is in accessible mode.

## Type

`Telerik.ReportViewer.Blazor.AccessibilityKeyMap`

## Examples

````JavaScript
<ReportViewer AccessibilityKeyMap="@(new AccessibilityKeyMap())" />
````

# The `ToString()` method of the ASP.NET Web Forms Report Viewer

Converts the HTML5 ReportViewer control to its HTML string representation with all configuration options and client-side initialization. <remarks>Generates the final HTML output by populating viewer options and using ReportViewerWriter to create the complete HTML5 viewer structure with JavaScript initialization.</remarks>

Returns: A complete HTML string containing the HTML5 viewer markup, scripts, and configuration for browser rendering.

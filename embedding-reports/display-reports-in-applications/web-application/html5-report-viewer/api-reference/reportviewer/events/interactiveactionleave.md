# interactiveActionLeave

Called when leaving an interactive action area.   Receives two parameters: `e` (the jQuery.Event object; `e.data.sender` is the report viewer instance)   and `args` (an object with properties: `element` (DOM element), `action` ({@link InteractiveAction})).

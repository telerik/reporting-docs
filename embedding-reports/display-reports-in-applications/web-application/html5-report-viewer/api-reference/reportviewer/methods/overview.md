# Methods of the HTML5 Report Viewer

| Method | Description |
| ------ | ------ |
| [bind(eventName, eventHandler)]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/methods/bind(eventname,-eventhandler)%})|Binds event handlers to the specified event.|
| [dispose()]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/methods/dispose()%})|Disposes the viewer instance, cleaning up resources and event handlers. After calling this method, the viewer instance should not be used. Stops sending keep alive requests to the server, if keep client alive was enabled.|
| [parametersAreaVisible(visible)]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/methods/parametersareavisible(visible)%})|Shows or hides the parameters area in the viewer UI.|
| [unbind(eventName, [eventHandler])]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/methods/unbind(eventname,-eventhandler)%})|Unbinds an event handler from a viewer event.|

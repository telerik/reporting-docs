# Read-Only Properties of the HTML5 Report Viewer

| Property | Description |
| ------ | ------ |
| [currentPage]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/read-only-properties/currentpage%})|Gets the viewer’s current page that is displayed.|
| [getReportParameters]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/read-only-properties/getreportparameters%})|Returns an object with properties, the names of which are the report parameters’ IDs, and values - the report parameters’ values. The values of multivalue parameters include their display and value members.|
| [isReportAutoRun]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/read-only-properties/isreportautorun%})|Returns whether auto-run is enabled for the report (i.e., whether the report runs automatically when parameters change).|
| [pageCount]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/read-only-properties/pagecount%})|Gets the total page count of the viewer’s currently displayed report.|
| [parameterEditors]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/read-only-properties/parametereditors%})|Allows the user to define array of custom editors for the report parameters.|
| [parameters]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/read-only-properties/parameters%})|Allows the user to define parameter options for the report parameters. Properties: editors (object).|
| [parametersAreaPosition]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/read-only-properties/parametersareaposition%})|The position of the parameters area. See {@link ParametersAreaPositions}.|
| [parametersAreaVisible]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/html5-report-viewer/api-reference/reportviewer/read-only-properties/parametersareavisible%})|Determines whether the viewer's parameters area is displayed if any parameter editor exists.|

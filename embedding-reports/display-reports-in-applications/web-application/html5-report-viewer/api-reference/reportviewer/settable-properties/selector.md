# selector

A selector used in conjunction with the data- attributes.

## Type

`string`

## Examples

````JavaScript
// Set a custom selector$("#reportViewer1").telerik_ReportViewer({  serviceUrl: "api/reports/",  reportSource: { report: "Dashboard.trdp" },  selector: "#myViewer"});
````

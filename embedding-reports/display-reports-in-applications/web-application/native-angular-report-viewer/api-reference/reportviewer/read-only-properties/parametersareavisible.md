# parametersAreaVisible

Controls the visibility of the parameters area if the report has parameters.

## Type

`boolean`

## Examples

````html
<reporting-angular-viewer
  [parametersAreaVisible]="parametersAreaVisible">
</reporting-angular-viewer>
````

````typescript
parametersAreaVisible: boolean = false;
````

# contentTabIndex

Sets the tab index for the report viewer content.

## Type

`number`

## Examples

````html
<reporting-angular-viewer
 [contentTabIndex]="contentTabIndex">
</reporting-angular-viewer>
````

````typescript
contentTabIndex: number = 1500;
````

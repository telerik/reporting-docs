# documentMapVisible

Controls the visibility of the document map panel if a document map is available.

## Type

`boolean`

## Examples

````html
<reporting-angular-viewer
 [documentMapVisible]="documentMapVisible">
</reporting-angular-viewer>
````

````typescript
documentMapVisible: boolean = false;
````

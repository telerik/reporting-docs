# Settable Properties of the Native Angular Report Viewer

| Property | Description |
| ------ | ------ |
| [authenticationToken]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/authenticationtoken%})|A bearer token to be added in the Authorization header of each request. Used for authentication with the reporting service.|
| [beginLoadReport]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/beginloadreport%})|Event emitted after the rendering of a report begins. Indicates that the report rendering process has started.  beginLoadReport|
| [contentTabIndex]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/contenttabindex%})|Sets the tab index for the report viewer content.|
| [documentMapVisible]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/documentmapvisible%})|Controls the visibility of the document map panel if a document map is available.|
| [enableSendEmail]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/enablesendemail%})|Enables or disables the send email functionality.|
| [exportDocumentReady]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/exportdocumentready%})|Event emitted after the viewer finishes exporting the report. Triggered when an export operation completes successfully.  exportDocumentReady|
| [exportStarted]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/exportstarted%})|Event emitted when an export operation is triggered. Fired at the start of any report export process.  exportStarted|
| [height]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/height%})|Sets the height of the report viewer container.|
| [interactiveActionExecuting]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/interactiveactionexecuting%})|Event emitted when an interactive action is being executed. Triggered for actions like drill-down, bookmarks, and hyperlinks.  interactiveActionExecuting|
| [keepClientAlive]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/keepclientalive%})|Determines whether to keep the client session alive. When true, prevents the session from timing out during long periods of inactivity.|
| [loadedReportChange]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/loadedreportchange%})|Event emitted when the loaded report changes. Triggered during navigation between different reports.  loadedReportChange|
| [navigateToReport]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/navigatetoreport%})|Event emitted when navigating to a different report. Triggered by interactive actions that load a new report, such as drill-through actions. The event handler receives an object of type 'ReportSourceOptions' containing the new report source options. The ReportSourceOptions type should be imported from '@progress/telerik-common-report-viewer'.  navigateToReport|
| [pageMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/pagemode%})|Sets the page display mode of the viewer.|
| [persistSession]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/persistsession%})|Determines whether to persist the viewer session across browser sessions.|
| [printDocumentReady]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/printdocumentready%})|Event emitted after the viewer finishes printing the report. Indicates that the print document is ready and the print operation is complete.  printDocumentReady|
| [printMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/printmode%})|Sets the print mode for the report viewer.|
| [printStarted]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/printstarted%})|Event emitted when printing starts. Triggered at the beginning of a print operation.  printStarted|
| [renderingStopped]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/renderingstopped%})|Event emitted when report rendering is stopped. Triggered when the user stops an ongoing rendering operation.  renderingStopped|
| [reportAutoRunOff]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/reportautorunoff%})|Event emitted when report auto-run is disabled. Triggered when the report requires user input for parameters before rendering.  reportAutoRunOff|
| [reportSource]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/reportsource%})|Specifies the report and initial report parameter values to be displayed. The report property can be a relative path to a physical file (TRDP/TRDX/TRBP) or an assembly qualified name for type definitions. The ReportSourceOptions type should be imported from '@progress/telerik-common-report-viewer'.|
| [scale]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/scale%})|Sets the scale factor for the report pages. The scale takes effect when the scaleMode is set to 'specific'.|
| [scaleMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/scalemode%})|Sets how the report pages are scaled.|
| [serviceType]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/servicetype%})|Specifies the type of service that the report viewer will connect to.|
| [serviceUrl]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/serviceurl%})|Sets the address of the Reporting REST Service. Required when serviceType is 'REST'.|
| [updateUI]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/updateui%})|Event emitted when the state of the viewer changes. This event is triggered during various viewer operations and state transitions.  updateUI|
| [viewMode]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/viewmode%})|Sets the view mode of the report viewer.|
| [width]({%slug telerikreporting/using-reports-in-applications/display-reports-in-applications/web-application/native-angular-report-viewer/api-reference/reportviewer/settable-properties/width%})|Sets the width of the report viewer container.|

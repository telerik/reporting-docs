# pageMode

Sets the page display mode of the viewer.

## Type

`PageMode`

## Examples

````html
<reporting-angular-viewer
  [pageMode]="pageMode">
</reporting-angular-viewer>
````

````typescript
pageMode: PageMode = 'singlePage';
````

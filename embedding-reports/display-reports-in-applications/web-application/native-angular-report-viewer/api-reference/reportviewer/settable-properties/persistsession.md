# persistSession

Determines whether to persist the viewer session across browser sessions.

## Type

`boolean`

## Examples

````html
<reporting-angular-viewer
  [persistSession]="persistSession">
</reporting-angular-viewer>
````

````typescript
persistSession: boolean = true;
````

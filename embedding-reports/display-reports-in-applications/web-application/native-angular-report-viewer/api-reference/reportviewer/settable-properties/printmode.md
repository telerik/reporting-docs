# printMode

Sets the print mode for the report viewer.

## Type

`PrintMode`

## Examples

````html
<reporting-angular-viewer
  [printMode]="printMode">
</reporting-angular-viewer>
````

````typescript
printMode: PrintMode = 'forcePDFFile';
````

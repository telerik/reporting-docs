# scale

Sets the scale factor for the report pages. The scale takes effect when the scaleMode is set to 'specific'.

## Type

`number`

## Examples

````html
<reporting-angular-viewer
 [scale]="scale">
</reporting-angular-viewer>
````

````typescript
scale: number = 1.5 // 150% scale
````

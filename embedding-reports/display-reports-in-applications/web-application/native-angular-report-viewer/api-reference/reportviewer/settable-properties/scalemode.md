# scaleMode

Sets how the report pages are scaled.

## Type

`ScaleMode`

## Examples

````html
<reporting-angular-viewer
 [scaleMode]="scaleMode">
</reporting-angular-viewer>
````

````typescript
scaleMode: ScaleMode = 'fitPageWidth';
````

# serviceType

Specifies the type of service that the report viewer will connect to.

## Type

`ServiceType`

## Examples

````html
<reporting-angular-viewer
 [serviceType]="serviceType">
</reporting-angular-viewer>
````

````typescript
serviceType: ServiceType = 'reportServer';
````

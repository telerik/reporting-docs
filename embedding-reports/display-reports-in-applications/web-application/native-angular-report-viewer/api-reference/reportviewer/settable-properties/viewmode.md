# viewMode

Sets the view mode of the report viewer.

## Type

`ViewMode`

## Examples

````html
<reporting-angular-viewer
 [viewMode]="viewMode">
</reporting-angular-viewer>
````

````typescript
viewMode: ViewMode = 'printPreview';
````

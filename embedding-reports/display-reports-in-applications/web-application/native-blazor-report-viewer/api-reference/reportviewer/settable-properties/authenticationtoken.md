# AuthenticationToken

Gets or sets the authentication (bearer) token that is passed in the Authorization header of the viewer's requests.

## Type

`System.String`

## Examples

````JavaScript
<ReportViewer 
    AuthenticationToken="your-auth-token">
</ReportViewer>
````

# ContentTabIndex

Gets or sets the tab index of the content within the Report Viewer. This property determines the order in which elements receive focus when navigating through the Report Viewer using the keyboard.

## Type

`System.Int32`

## Examples

````JavaScript
<ReportViewer 
    ContentTabIndex="1500">
</ReportViewer>
````

# CurrentCulture

Gets or sets the current culture used by the ReportViewer.

## Type

`System.Globalization.CultureInfo`

## Examples

````JavaScript
<ReportViewer 
    CurrentCulture="@CultureInfo.InvariantCulture">
</ReportViewer>
````

# KeepClientAlive

Gets or sets a value indicating whether the client will be kept alive. <p> When set to true a request will be sent to the server to stop the client from expiring, determined by the ClientSessionTimeout server configuration</p> <p> When set to false, the client will be left to be expired</p>

## Type

`System.Boolean`

## Examples

````JavaScript
<ReportViewer 
    KeepClientAlive="true">
</ReportViewer>
````

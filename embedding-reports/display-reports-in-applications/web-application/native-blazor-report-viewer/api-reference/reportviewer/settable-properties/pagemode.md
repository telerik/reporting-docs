# PageMode

Sets if the report is displayed in Single page or Continuous scroll mode.

## Type

`Telerik.ReportViewer.BlazorNative.PageMode`

## Examples

````JavaScript
<ReportViewer 
    PageMode="@PageMode.SinglePage">
</ReportViewer>
````

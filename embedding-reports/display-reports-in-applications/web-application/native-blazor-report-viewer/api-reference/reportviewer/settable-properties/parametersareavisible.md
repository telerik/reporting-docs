# ParametersAreaVisible

Gets or sets a value indicating whether the viewer's parameters area is initially visible.

## Type

`System.Boolean`

## Examples

````JavaScript
<ReportViewer 
    ParametersAreaVisible="true">
</ReportViewer>
````

# ReportVersionMismatchErrorMessage

Represents the error message used to indicate a version mismatch between the viewer and the Reporting REST Service / Report Server.

## Type

`System.String`

## Examples

````JavaScript
<ReportViewer 
    ReportVersionMismatchErrorMessage="The viewer and REST Service versions do not match.">
</ReportViewer>
````

# ServiceType

Specifies the type of service the ReportViewer is using.

## Type

`Telerik.ReportViewer.BlazorNative.ReportViewerServiceType`

## Examples

````JavaScript
<ReportViewer 
    ServiceType="@ReportViewerServiceType.REST">
</ReportViewer>
````

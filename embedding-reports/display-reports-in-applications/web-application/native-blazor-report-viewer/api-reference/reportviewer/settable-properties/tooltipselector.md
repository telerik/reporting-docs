# TooltipSelector

Gets or sets the tooltip selector for the report viewer. This property allows you to define a specific selector for tooltips in the report viewer, enabling customized hover information for report elements.

## Type

`System.String`

## Examples

````JavaScript
<ReportViewer 
    TooltipSelector=".custom-tooltip">
</ReportViewer>
````

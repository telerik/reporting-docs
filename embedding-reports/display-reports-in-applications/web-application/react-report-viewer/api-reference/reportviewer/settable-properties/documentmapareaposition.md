# documentMapAreaPosition

Specifies where the Document Map should be displayed. Available values: "RIGHT", "LEFT". Default value is "LEFT".

## Type

`string`

## Examples

````JavaScript
// Place document map on the right<TelerikReportViewer  serviceUrl="api/reports/"  reportSource={{ report: "Dashboard.trdp" }}  documentMapAreaPosition="RIGHT"/>
````

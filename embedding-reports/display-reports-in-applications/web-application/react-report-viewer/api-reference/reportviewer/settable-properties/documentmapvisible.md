# documentMapVisible

Determines whether the viewer's document map is displayed if any bookmark is defined.

## Type

`boolean`

## Examples

````JavaScript
// Show document map<TelerikReportViewer  serviceUrl="api/reports/"  reportSource={{ report: "Dashboard.trdp" }}  documentMapVisible={true}/>
````

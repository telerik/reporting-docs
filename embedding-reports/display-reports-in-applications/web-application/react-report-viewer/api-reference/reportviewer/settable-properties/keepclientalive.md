# keepClientAlive

Determines whether the client will be kept alive. Default value is true.

## Type

`boolean`

## Examples

````JavaScript
// Keep client alive<TelerikReportViewer  serviceUrl="api/reports/"  reportSource={{ report: "Dashboard.trdp" }}  keepClientAlive={true}/>
````

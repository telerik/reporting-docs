# scale

Sets the scale factor for the report pages. The scale takes effect when scaleMode is set to "SPECIFIC". Default value is 1.0 (100%).

## Type

`number`

## Examples

````JavaScript
// Set zoom to 150%<TelerikReportViewer  serviceUrl="api/reports/"  reportSource={{ report: "Dashboard.trdp" }}  scale={1.5}  scaleMode="SPECIFIC"/>
````

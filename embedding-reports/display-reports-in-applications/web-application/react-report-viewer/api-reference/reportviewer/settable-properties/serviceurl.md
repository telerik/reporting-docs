# serviceUrl

Sets the address of the Report REST Service. Required if reportServer is not provided.

## Type

`string`

## Examples

````JavaScript
// Set the service URL for the report viewer<TelerikReportViewer  serviceUrl="api/reports/"  reportSource={{ report: "Dashboard.trdp" }}/>
````
